Тестирование

Тестирование — это процесс проверки корректности работы кода.

Цель тестирования — убедиться, что программы, функции или отдельные части вашего кода (модули) работают ожидаемым образом, находить ошибки (баги) до того, как они попадут в рабочее приложение.

  1. UNIT тесты — это код, который проверяет работу определённого модуля.

  2. Интеграционные тесты — это тесты, в которых проводится проверка взаимосвязи разных модулей в изолированном окружении.

  3. Функциональное тестирование — проверка конкретных функций приложения с точки зрения пользователя.

  4. End-to-End тесты (E2E тесты) — это тесты, которые тестируют целые пользовательские сценарии.

    5. Основные инструменты для тестирования:

  • Jest — современный тестовый фреймворк, часто используется с React.

  • Mocha и Chai — популярная связка для тестирования.

  • Jasmine — тестовый фреймворк без необходимости в сторонних assertion библиотек.

  • Cypress, Puppeteer, Playwright — для автоматизации браузерных и E2E-тестов

Фреймворк Jest

Jest — это фреймворк для тестирования JavaScript с акцентом на простоту.

Работает с проектами, использующими Babel, TypeScript, Node, React, Angular, Vue и многое другое!


Установка Jest

npm i -D jest

В "scripts" файла package.json прописать:

"test": "jest",

Установка Jest для TypeScript

npm i -D ts-jest // основной пакет

npm i -D @types/jest // дополнительный пакет

// в tsconfig.json в раздел "compilerOptions"
"types": [
  "jest"
],

// в корень проекта добавить файл jest.config.cjs
// в нём прописать настройки:
module.exports = {
  preset: "ts-jest", 
}
// они обеспечивают возможность использования инструментов для написания тестов в коде

// прописать команду:
npx test 
// и написать ответ y или yes на вопрос

  4. Методы toBeCreaterThanOrEqual и toBeLessThanOrEqual для проверки диапазонов

    // task_03.js
function randomInt(min, max) {
  let rand = min + Math.random() * (max + 1 - min);
  return Math.floor(rand);
}

module.exports = randomInt;

// random.test.js
const randomInt = require('./task_03');

describe("Тестирование случайных чисел", () => {
  testCases = [
    {
      from: 100,
      to: 110,
    },
    from: 200,
    to: 201,
  ];
  testCases.forEach(test => {
    it(
      `from: ${test.form} to: ${test.to}`,
      () => {
        const res = randomInt(test.from, test.to);
        console.log(res);
        expect(res).toBeCreaterThanOrEqual(test.form); // Ожидание, что res будет равен или больше, чем test.from
        expect(res).toBeLessThanOrEqual(test.to); // Ожидание, что res будет равен или меньше, чем test.to
      }
    );
  });
});
